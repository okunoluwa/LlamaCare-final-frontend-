LlamaCare - Patient Assistance System
A comprehensive web-based platform helping patients navigate their healthcare journey from pre-consultation through post-operative recovery.

 Features
Patient Registration & Login - Secure account management

Pre-Consultation Intake - 7-question medical assessment with progress tracking

Consent Form Explanation - Medical jargon simplified in 11 South African languages

Pre-Op Preparation - Personalized surgery information and reminders

Anxiety Reduction - Guided breathing, audio reassurance, cultural support

Post-Op Guidance - Recovery timeline, symptom tracking, wound care

Hospital Dashboard - Staff interface with voice assistant

Dark/Light Mode - Persistent theme preference

 Quick Start
Clone the repository

Open HomePage.html in your browser

Register a new account or use demo accounts

Navigate through the patient journey

No build process or dependencies required!
